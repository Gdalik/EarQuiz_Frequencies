Диапазон делится на фрагменты одинаковой длительности от точки start (сколько укладывается по длине целиком)
??При перемотке << >> [Возможно, не нужна] позиция в начало ближайшего фрагмента

Режим Preview:
Прослушать аудио, установить точки start/end, изменить длительность фрагментов
Курсор ставится в любую точку
Воспроизведение -- в пределах диапазона
Кнопка Stop -- переход в начало диапазона
Изменение позиции эквалайзера запускает режим Learn

Режим Learn:
Установка start/end не допускается, изменение длительности фрагментов выключено, Pause выключена
Переключение на другой трек включает режим Preview
Курсор при воспроизведении автоматически перемещается в начало текущего фрагмента
Кнопка Stop -- переход в начало диапазона
Частоты переключаются сами (цикл снизу вверх по частотам)
Пользователь может самостоятельно активизировать прослушивание на нужной частоте, просто потянув за соответствующий slider
(или выставить 2 slider-а -- в dual-band упражнениях). При последующем нажатии на Play будет активизирована следующая по
порядку частота.

Режим Test:
Установка start/end не допускается, изменение длительности фрагментов выключено, перемотка выключена, Pause выключена
самостоятельное перемещение курсора выключено, переход на другой трек/переключение на другой источник аудио невозможен.
Кнопки Start/Stop/Pause -- работают в пределах текущего примера.
Ответ засчитывается при изменении позиции слайдера (или 2 слайдеров -- в dual-band упражнениях)

*.afab : Файл, сохраняющий данные по рабочим диапазонам аудиофайлов. JSON: Hash: [Filename, A: int/float, B: int/float, Slice_length].

По завершению теста все кнопки режима отжаты, воспроизведение отключено. Нажатие приводит к обнулению результатов.

Контекстное меню эквалайзера: Playback order (Ascending / Descending / Random no-repeat| Each band boosted, then cut / All bands boosted, then all bands cut ),
View Eq Settings
Меняется в зависимости от паттерна.
В 2-полосном режиме -- только случайным образом, чтобы не утомлять слух.

Ожидаемый диапазон результатов:
- для октавных полос (1 фильтр): 85-95;
- для 1/3 октавных полос (1 фильтр): 75-90
- для октавных полос (2 фильтра): 70-85
- для 1/3 октавных полос (2 фильтра): 65-80

Минус 10 баллов от нижней границы -- граница "провала" теста
